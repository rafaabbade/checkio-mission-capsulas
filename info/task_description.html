Enunciado

Fan Chi'ih trouxe cápsulas mágicas da China, cada uma com um ciclo de produção específico que produz uma moeda de ouro no último dia do ciclo. Ao ativar todas as cápsulas simultaneamente, ele deseja saber quantos dias são necessários para acumular pelo menos F moedas de ouro, dado o ciclo de produção de cada cápsula.

A imagem acima mostra os ciclos de produção de cada moeda (3, 7, 2).

Entrada Uma lista onde o primeiro elemento é um inteiro N representando o número de cápsulas, o segundo é um inteiro F representando o número desejado de moedas, seguido por uma lista de N inteiros representando os ciclos de produção de cada cápsula.

Saída: Um inteiro representando o número mínimo de dias necessários para produzir pelo menos F moedas.

Example:

{% endif %}
